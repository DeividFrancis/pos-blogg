# BLOGG

Projeto simples de uma api para um blog para treinar o uso:

- Prisma
- Typescript
- Postgress
- Express

## DEV

- Banco de dados

```sh
$ docker-compose up
```

- Aplicacao

1. troca o nome do aquivo `.env.example` para `.env`;

2. rode o comando `npm run dev`
